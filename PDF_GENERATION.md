# PDF Generation Instructions

To generate a PDF of the Physical AI Book:

1.  **Run the site**: `npm start`
2.  **Open the browser**: Navigate to `http://localhost:3000/docs/01-foundations`
3.  **Print**: Press `Ctrl+P` (or `Cmd+P`)
4.  **Destination**: "Save as PDF"
5.  **Settings**: 
    - Margins: Minimum
    - Background graphics: Checked (if needed for diagrams)
    - Headers and footers: Unchecked

*Note: Automated PDF generation via Puppeteer can be added in future iterations.*
